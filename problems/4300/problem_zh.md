## 题目描述

给定一个长度为 $n$ 的序列 $a_{1\cdots n}$，求它满足 $\forall i\in[1,n),b_i \&b_{i+1}\not =0$ 的子序列 $b$ 的最大长度，其中 $\&$ 是按位与操作。

## 输入格式

第一行一个整数 $n$。

第二行 $n$ 个整数 $a_{1\cdots n}$。

## 输出格式

一行一个整数表示子序列 $b$ 的最大长度。

```input1
3
1 2 3
```

```output1
2
```

## 数据规模与约定

对于 $100\%$ 的数据，$1\leq n\leq 10^5$，$a_i\leq 2\times 10^9$。
