## 题目描述

给你一个序列 $s$，你把这个序列的所有不同排列按字典序排列后，求 $s$ 的排名 $\bmod m$。

## 输入格式

序列的长度 $n < 3 \times 10^5$ 和 $m$。

$n$ 个数，代表序列 $s$。

## 输出格式

排名 $\bmod m$。

```input1
4 1000
2 1 10 2
```

```output1
5
```

## 数据规模与约定

$2 \le m \le 10^9,1 \le S_i \le 3 \times 10^5$。

## 样例说明

所有字典序比给定序列小的排列有：

$ (1,2,2,10), (1,2,10,2), (1,10,2,2), (2,1,2,10)$。
