## 题目描述

## 输入格式

输入文件的第一行包含一个整数 $N$，代表每队的人数。
接下来的 $N$ 行，每行一个整数，表示你的每个队员的能力值。
接下来的 $N$ 行，每行一个整数，依次表示对方团队的每个队员的能力值。
最后一行包含一个整数，代表 $L$。

## 输出格式

输出文件仅包含一个实数，表示你最大可能获得的分数。答案四舍五入保留到小数点后六位。

## 样例

```input1
4 
2239 
2412 
2399 
2267 
2534 
2429 
2340 
2389 
100
```

```output1
1.483574
```
