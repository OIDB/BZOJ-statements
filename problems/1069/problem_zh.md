## 题目描述

在某块平面土地上有 $N$ 个点，你可以选择其中的任意四个点，将这片土地围起来，当然，你希望这四个点围成
的多边形面积最大。

## 输入格式

第 $1$ 行一个正整数 $N$，接下来 $N$ 行，每行两个数 $x,y$，表示该点的横坐标和纵坐标。

## 输出格式

最大的多边形面积，答案精确到小数点后三位。

```input1
5
0 0
1 0
1 1
0 1
0.5 0.5
```

```output1
1.000
```

## 提示

对所有的测试点： $4\le N\le 2\times 10^3$, $|x|,|y|\le 10^5$。
