## 题目描述

平面坐标系 $\text{xOy}$ 内，给定 $n$ 个顶点 $(x_i , y_i)$ 。对于顶点 $u,v$，$u$ 与 $v$ 之间的距离 $d$ 定义为 $|x_u - x_v| + |y_u - y_v|$。 

你的任务就是求出这 $n$ 个顶点的最小生成树。 

## 输入格式

第一行一个正整数 $n$，表示定点个数。

接下来 $n$ 行每行两个正整数 $x,y$，描述一个顶点。 

## 输出格式

只有一行，为最小生成树的边的距离和。 

## 样例输入

```plain
4
1 0
0 1
0 -1
-1 0
```

## 样例输出

```plain
6
```

## 数据规模与约定

对于 $100\%$ 的数据，$n\leq 5\times 10^4$，$0\leq x,y\leq 1\times 10^5$。
