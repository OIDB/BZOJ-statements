## 题目描述

给定一个长度为 $n$ 的整数序列 $x_i$，确定一个二元组 $(b, k)$ 使得 $s=\sum(k\times i+b-x_i)^2$（$i\in [0,n-1]$）最小。

## 输入格式

第一行一个整数 $n$。

第二行接下来 $n$ 个整数，为 $x_i$。

## 输出格式

包含一行两个实数为二元组 $(b,k)$。

```input1
4
0 0 0 0 
```

```output1
0 0
```
## 数据规模与提示

对于 $100\%$ 的数据，$n\le 10^6$，每个 $x_i$ 的绝对值不超过 $10^8$。

你的 $s$ 不超过标准答案的 $100.01\%$ 时，被认为是可以的。
