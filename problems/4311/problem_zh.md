## 题目描述

你要维护一个向量集合，支持以下操作：

- `1 x y`，插入一个向量 $(x,y)$；

- `2 i`，删除插入的第 $i$ 个向量；

- `3 x y`，查询当前集合与 $(x,y)$ 点积的最大值是多少。如果当前是空集输出 $0$。

## 输入格式

第一行输入一个整数 $n$，表示操作个数。

接下来 $n$ 行，每行一个操作。

保证一个向量只会被删除一次，不会删没有插入过的向量。

## 输出格式

对于每条询问，输出一行一个答案。

```input1
5
1 3 3
1 1 4
3 3 3
2 1
3 3 3
```

```output1
18
15
```

## 数据规模与约定

对于 $100\%$ 的数据，$1\leq n\leq 2\times 10^5$，$1\leq x,y\leq 10^6$。
