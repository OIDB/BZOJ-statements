## 题目描述

给出一个 $n\times n$ 的数字矩阵。

你从左上角出发，可以向右走，或者向下走，不能出界，你所移动的长度为当前你所在的格子的数字的值。

## 输入格式

先给出一个数字 $n$。

再给出 $n\times n$ 的数字矩阵，每个数字 $a_{i,j}$ 的值在 $10$ 以内。

## 输出格式

从左上角走到右下角有多少种走法。


```input1
4
2 3 3 1
1 2 1 3
1 2 3 1
```


```output1
3
```

## 数据规模与约定

对于 $100\%$ 的数据，满足 $n\leq 100$，$a_{i,j}\leq 10$。
