## 题目描述

给出 $n$ 个二维平面上的点，保证任意三点不共线。

求任意三点组成的三角形的最大外接圆半径。

## 输入格式

第一行有一个整数 $n$，表示点的个数。

接下来 $n$ 行，每行两个整数，为点的坐标 $(x_i,y_i)$。

## 输出格式

你所得到的圆的半径（精确到小数点后 $3$ 位）。


```input1
11
4
0 0
1 0
1 1
3 1 
```

```output1
3.536
```

## 样例解释

说明：选 $(0,0),(1,0),(3,1)$ 三点。

## 数据规模与约定

对于 $100\%$ 的数据，$1 \leq n \leq 655$，$0 \leq x,y \leq 32767$。
