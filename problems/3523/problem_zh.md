## 题目描述

有 $n$ 种颜色的砖块，第 $i$ 种颜色的砖块有 $a_i$ 个，你需要把他们放成一排，使得相邻两个砖块的颜色不相同，限定第一个砖块的颜色是 $start$，最后一个砖块的颜色是 $end$，请构造出一种合法的方案或判断无解。

## 输入格式

第一行 $3$ 个数，$n,start,end$。

第二行 $n$ 个数，第 $i$ 个数表示 $a_i$。

## 输出格式

令 $m = \sum_{i = 1}^{n} a_i$。

如果有解输出 $m$ 个数。

无解输出 $0$。

```input1
3 3 1
2 3 3
```
```output1
3 2 1 3 2 3 2 1
```

## 提示

$n,m \le 10^6,~1 \le start,end \le n$
