## 题目描述

给定 $n$ 个数 $a_{1\cdots n}$，对序列 $b_{1\cdots n}$ 计数，要求其满足：

1. 任意 $b_i\le a_i$；

2. $\oplus_{i=1}^nb_i=0$；

3. $\sum_{i=1}^{n}b_i>0$。

## 输入格式

每一行给出数字 $n$。

下面一行给出 $n$ 个数字 $a_{1\cdots n}$。

## 输出格式

一行一个整数表示答案。

```input1
3
2 1 3
```

```output1
5
```

## 样例解释

满足条件的 $b$ 有：$(0,1,1)$，$(1,0,1)$，$(1,1,0)$，$(2,0,2)$，$(2,1,3)$。

## 数据规模与约定

对于 $100\%$ 的数据，$2\leq n\leq 50$，$1\leq a_i\leq 2^{32}-1$。
