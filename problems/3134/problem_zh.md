## 题目描述

一个数是非回文数当且仅当不包含长度大于 $1$ 的回文数。比如 $16276$ 是非回文数，而 $17276$ 因为含有 $727$ 而不是。
求区间内有多少个非回文数。

## 输入格式

一行两个整数 $l,r$ 表示区间。

## 输出格式

非回文数数量。



```input1
123 321
```



```output1
153
```



```input2
123456789 987654321
```



```output2
167386971
```

## 数据规模与约定

对于 $100\%$ 的数据，$0 \le l \le r \le 10^{18}$。
