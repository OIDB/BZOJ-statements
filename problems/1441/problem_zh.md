## 题目描述

给出 $n$ 个数 $a_{1\cdots n}$，求一组整数序列 $x_{1\cdots n}$ 使得 $S=\sum a_ix_i>0$ 且最小。

## 输入格式

第一行一个整数 $n$。

第二行 $n$ 个整数，表示 $a_{1\cdots n}$。

## 输出格式

一行一个正整数，表示最小的 $S$。

```input1
2
4059 -1782
```

```output1
99
```

## 数据规模与约定

对于 $100\%$ 的数据，$1\leq n\leq 20$，$|a_i|\leq 10^4$。
