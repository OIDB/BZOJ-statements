## 题目描述
$S$ 国是一个爱好数字的国度，总有些人会夜以继日的数着数字以占卜凶吉，凡是数字在十进制表示之下有连续 $k$ 个 $4$ 或 $k$ 个 $7$ 相连则被认为是幸运的。他们看遍了 $1$ 到 $n$ 之间的所有数，想考一考你。因为他们预先知道答案，所以他们只需你告诉他们幸运的数字 $\bmod 10^{9}+7$ 就行。
## 输入格式
第一行一个整数 $k$；

第二行一个整数 $n$。
## 输出格式
一个数，表示答案。

```input1
1
100
```

```output1
36
```

## 数据范围
设 $t$ 为 $n$ 的位数
对于 $100\%$ 的数据$1\leq k\leq t\leq 10^6$
